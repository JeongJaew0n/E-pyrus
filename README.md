# E-pyrus

소설 사이트 챕터 크롤러.

## Chrome Extension (메인) — `cheese-extension/`

### 설치

1. `chrome://extensions/` → 개발자 모드 ON
2. "압축해제된 확장 프로그램을 로드합니다" → `cheese-extension/` 폴더 선택

### 사용법

1. 대상 사이트 방문하여 Cloudflare 통과
2. 확장 아이콘 클릭 또는 사이트에서 🧀 버튼 클릭
3. URL 입력 → 크롤링 시작

## Electron App (레거시)

Node.js `20.18.1` 이상

```bash
npm install
npm start
```
